# ⚽ Allsvenskan 2023 - Team Performance Analysis

EDA project analyzing Swedish football league (Allsvenskan) 2023 season statistics.

## 📊 What's Inside
- Team standings and points analysis
- Win rate vs goal difference comparison
- Goals scored vs conceded visualization
- Goals per game ranking

## 🔍 Key Findings
- 🏆 Champion: Malmö FF (66 points, 66.7% win rate)
- ⚽ Top scorer: Malmö FF (58 goals)
- 🛡️ Best defense: Malmö FF (only 28 goals conceded)
- 😬 Worst team: GIF Sundsvall (15 points, 10% win rate)
- 📈 League average goals per game: 1.19

## 🛠️ Tools Used
- Python, Pandas, Matplotlib, Seaborn

## ⚠️ Note
Data is manually compiled for educational purposes.
